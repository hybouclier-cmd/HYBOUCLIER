# HYBOUCLIER™ — Roadmap officielle

## 🎯 Vision générale
HYBOUCLIER™ est un réseau social citoyen, local-first, éthique et non-commercial.  
Cette roadmap présente les étapes prévues pour le développement du projet.

---

## ✅ Version 0.2 — (Complétée)
- Structure du dépôt GitHub
- README de base
- Avis légal dans la description
- Fichiers de contribution (CONTRIBUTING.md, HEART.md, TESTING.md)
- Nettoyage des dossiers inutiles

---

## 🚧 Version 0.3 — (En cours)
- Organisation des dossiers (backend, frontend, docs)
- Début de la documentation technique
- Définition des modules principaux
- Préparation de l’architecture local-first

---

## 🔮 Version 1.0 — (Objectif)
- Première version fonctionnelle
- Interface minimaliste
- Système de comptes local-first
- Publication publique du projet

---

## ❤️ Valeurs fondamentales
- Éthique  
- Non-commercial

- Respect de la vie privée  
- Simplicité
