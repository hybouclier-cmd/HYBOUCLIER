    À propos

    HYBOUCLIER est un écosystème web et app conçu pour rester simple, léger et évolutif. Le site offre des ressources externes et oriente vers l'app locale qui gère l'inscription et la connexion sans serveur actif.

    Le projet est préparé pour un futur serveur local et une future base de données, mais aujourd'hui tout est désactivé et fonctionne avec un stockage local minimal.

    État actuel

    • Inscription / connexion active via stockage local.
    • Stockage léger local : localStorage + JSON.
    • Option Serveur Local : visible sur le site, désactivée.
    • Option Base de données : prête dans le code, désactivée.
    • Pas de collecte externe. Pas de cloud. Pas de serveur actif.

    HYBOUCLIER fonctionne comme un seul écosystème : le site présente, l'app réalise. Tout est prêt pour évoluer sans compromettre le modèle local-first.
